# Digital Communication System Simulation

این پوشه شامل یک پروژه کامل در زمینه سیستم‌های مخابراتی دیجیتال است که بر اساس مقاله اصلی [Foundational Techniques for Wireless Communications](https://arxiv.org/pdf/2103.03970) پیاده‌سازی شده است.

## محتویات پوشه

- **ترجمه مقاله**: متن اصلی مقاله به فارسی ترجمه و خلاصه‌سازی شده است.
- **پیاده‌سازی مقاله**: شبیه‌سازی مدولاسیون‌های BPSK و QPSK، کد تصحیح خطای Hamming و Equalizer انجام شده است.
- **نمودارها و نتایج**: نتایج شبیه‌سازی شامل نرخ خطای بیت (BER) در شرایط مختلف کانال و مقایسه مدولاسیون‌ها به همراه نمودارها ارائه شده است.
- **سورس‌کدها**: تمام کدهای Python و فایل‌های مرتبط با شبیه‌سازی موجود است.
- **تحلیل و نتیجه‌گیری**: تحلیل نتایج و توضیح اثرات نویز، کد تصحیح خطا و Equalizer روی کیفیت انتقال داده ارائه شده است.

---

این پروژه به عنوان نمونه‌ای کاربردی از مفاهیم مخابرات دیجیتال و شبیه‌سازی کانال نویزی، مناسب برای درس‌های مرتبط با مدارها و سیستم‌های مخابراتی می‌باشد.
